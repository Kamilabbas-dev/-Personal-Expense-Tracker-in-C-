# -Personal-Expense-Tracker-in-C-
## Overview This project is a console-based Personal Expense Tracker developed in C++. It helps users manage income, expenses, savings, and budgeting records efficiently.
## Author
Kamil Abbas
## Features
- Add daily expenses
- Track income
- View total balance
- Expense categorization
- File handling support
- Simple console interface

- Arrays & Functions
